Repo for oom course

## Development

### Installation

- install [Nodejs](https://nodejs.org/en/)
- install a modern browser (e.g Chrome)

### Client

```bash
npm install
npm start
```

### Server

```bash
npm install
npm run server
```

## Deployment

### Client

```bash
npm install
npm run build
```

then deploy `build/` directory with Ngnix server etc.

### Server

```bash
npm install
npm run server
```
